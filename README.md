# Movie Recommender (PyTorch)

Небольшой учебный проект рекомендательной системы на PyTorch.  
На синтетическом датасете показываю полный путь: данные → матрица user–item → модель → обучение → предсказания.

## 1. Данные

В проекте используется маленький искусственный датасет рейтингов фильмов:

- 5 пользователей
- 4 фильма (ID 101–104)
- рейтинг от 1.0 до 5.0

Таблица строится прямо в коде с помощью `pandas.DataFrame`, затем превращается в матрицу пользователь–фильм (user–item matrix) с помощью `pivot()`.

## 2. Матрица user–item

Из таблицы рейтингов формируется матрица:


- строки — пользователи (`user_id`)
- столбцы — фильмы (`movie_id`)
- значения — рейтинг (0.0, если пользователь фильм не оценивал)

Эта матрица используется как вход для модели и как источник данных для датасета в PyTorch.

## 3. Модель

Модель `SimpleRec` реализована на PyTorch и представляет собой простую схему collaborative filtering:

- обучаемый эмбеддинг для пользователей (`nn.Embedding`)
- обучаемый эмбеддинг для фильмов (`nn.Embedding`)
- конкатенация векторных представлений пользователя и фильма
- полносвязный слой (`nn.Linear`)
- `sigmoid` на выходе для получения оценки от 0 до 1

Выход можно интерпретировать как вероятность того, что пользователю понравится фильм.

## 4. Обучение

Обучение реализовано вручную, без высокоуровневых обёрток:

- оптимизатор: `Adam`
- функция потерь: `BCELoss`
- данные подаются через `DataLoader` и класс `RecDataset`
- на каждой эпохе выводится значение функции потерь

Пример вывода обучения:
Training...
Epoch 1, loss = 1.9229
Epoch 2, loss = 1.6326
Epoch 3, loss = 1.3711
Epoch 4, loss = 1.1361
Epoch 5, loss = 0.9273


Loss постепенно уменьшается, что показывает, что модель подстраивается под данные.

## 5. Предсказания
  После обучения модель используется, чтобы получить предсказания «оценки» для конкретного пользователя:

Predicted scores for user #1:
Movie #101: 0.732
Movie #102: 0.898
Movie #103: 0.610
Movie #104: 0.720

Чем ближе значение к 1.0, тем выше рекомендация.
В примере для пользователя #1 самым подходящим фильмом модель считает фильм с ID 102.

## 6. Как запустить
 1. Открыть проект в Google Colab или локально.
 2. Установить зависимости (для Colab достаточно выполнить ячейку): !pip install torch torchmetrics scikit-learn pandas implicit
 3. Последовательно выполнить ячейки:
   1. Создание датасета и матрицы user–item
   2. Определение модели SimpleRec
   3. Цикл обучения (5 эпох)
   4. Блок с предсказаниями для выбранного пользователя

Код рассчитан на запуск «как есть» без дополнительных файлов и датасетов.
